1. Есть ли разница между $("h1 div"); и $("h1").$("div"); - может ли привести к тому что, поиск найдёт
разные элементы?
Если может - приведите пример, когда.

При поиску по селектору $("h1 div") - два параметра написаны через пробел, значит найдётся первый элемент,
который имеет сразу два эти параметра.

При поиску по селектору $("h1").$("div") - вначале найдётся первый элемент с параметром "h1", а затем среди его
"детей" найдётся первый элемент с параметром "div".

Может.
При $("h1 div") найдётся class="class2".
При $("h1").$("div") не найдётся class="class2".

<h1>
    class="class1"
</h1>
<h1>
    <div>
        class="class2"
    </div>
</h1>
